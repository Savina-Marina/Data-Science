# Python OOP (module 01)

Практическая работа по объектно-ориентированному программированию на Python.  
Цель — показать умение проектировать код, работать с классами, структурировать логику и избегать "data science style" хаоса.

---

## Общий стек

- Python 3
- ООП (классы, методы, наследование)
- Работа с файлами (CSV)
- Обработка ошибок
- Логирование
- Разделение кода (модули + конфиг)
- Базовая интеграция с внешними API (Telegram)
